# Zorvixa Launcher

>  **Important:** Read this before running anything.

## How to Use

To safely launch Zorvixa, follow these steps:

### Step 1: Run the Executor Script

1. Locate `ZorvixaExecutor.ps1` in your folder.  
2. **Right-click** the file.  
3. Select **Run with PowerShell**.  

> This script will automatically launch `Zorvixa.reworked NEW SHIT.exe` for you.

### Step 2: Safety Features

`ZorvixaExecutor.ps1` includes built-in safety measures to prevent:  
- Crashes  
- RAM flooding  
- Unstable execution  

##  Do NOT Do This

- Do **not** run `Zorvixa.reworked NEW SHIT.exe` manually  
- Running the `.exe` first may cause:
  - High memory usage  
  - Crashes  
  - Unpredictable behavior  

## Summary

 Run `ZorvixaExecutor.ps1` via **Right-click > Run with PowerShell**  
 Do NOT run the `.exe` manually
